Часто задаваемые вопросы

1) Для Добавления клиента/преподавателя необходимо нажать кнопку "Добавить клиента" В открывшемся окне требуется заполнить основные данные и нажать кнопку "Сохранить".

2) Для Изменения клиента/преподавателя необходимо нажать кнопку с изображением карандаша напротив выбранной записи. В открывшемся окне пользователь изменяет нужные данные и нажимает кнопку "Сохранить"

3) Удаление клиента/преподавателя происходит нажатием кнопки с изображением мусорного ведра напротив выбранной записи. В появившемся окне, для завершения процесса, нажать кнопку "Подтвердить"

1) Чтобы Добавить курс необходимо нажать кнопку "Добавить новый курс". В открывшемся окне требуется заполнить основные данные и нажать кнопку "Сохранить". ВНИМАНИЕ! Создании курса требуется назначить преподавателя. Без преподавателя курс не может быть сохранен в системе!

2) Для Изменения записи о курсе необходимо нажать кнопку с изображением карандаша напротив выбранной записи. В открывшемся окне пользователь изменяет нужные данные и нажимает кнопку "Сохранить"

3) Удаление клиента/преподавателя происходит нажатием кнопки с изображением мусорного ведра напротив выбранной записи. В появившемся окне, для завершения процесса, нажать кнопку "Подтвердить"

1) Чтобы Добавить заказ необходимо нажать кнопку "Добавить новый заказ". В открывшемся окне требуется заполнить основные данные и нажать кнопку "Сохранить".

2) Для Изменения записи о заказе необходимо нажать кнопку с изображением карандаша напротив выбранной записи. В открывшемся окне пользователь изменяет нужные данные и нажимает кнопку "Сохранить"

3) Удаление заказа происходит нажатием кнопки с изображением мусорного ведра напротив выбранной записи. В появившемся окне, для завершения процесса, нажать кнопку "Подтвердить"

На странице "Аналитика":

Графики "Заказы в месяц", "Зарплаты в год", "Продажи в год" и "Оборот" представляют доходы и расходы предприятия от деятельности преподавательского состава

Графики "Ожидаемое количество клиентов" и "Количество клиентов" помогают выявить текучку клиентов и проанализировать их оптимальное количество, для увеличения прибыли предприятия

График "Прибыль/Убытки" обощает все финансовые показатели и иллюстрирует общую ситуацию о финансовых делах предприятия

На странице "Главная":

График "Выручка в этом году" показывает доход предприятия от продаж онлайн-курсов. "Заказов"

На странице "Настройки":

Пользователь программного средства может увидеть собственные данные: имя, фамилия, отчество, имя пользователя программного средства, почта и пароль

Редактирование данных пользователя осуществляется нажатием кнопки "Редактировать" и последующим изменением данных. По нажатию кнопки "Сохранить" происходит сохранение измененией в системе

В следующем обновлении:

1) Будет добавлено изменение языка интерфейса

2) Будет добавлено изменение темы программного средства

3) Будет добавлено управление уведомлениями программного средства
